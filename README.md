# go-xkcd
CLI interface to read XKCD comics, implemented in Go language

# Usage
```
go-xkcd - Read XKCD from command line.

usage: go-xkcd [<args>]
  -n int
    	Get a specific comic number
  -o string
    	Output format (text/json) (default "text")
  -r	Get a random comic
```
